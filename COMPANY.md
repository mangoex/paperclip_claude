# Humanio Marketing

Agencia de marketing digital enfocada en negocios locales en México.
Operamos un pipeline automatizado de prospección, calificación SEO, diseño web, outreach comercial y cierre de ventas.

## Pipeline

```
Scout → Qualifier → WebDesigner → Outreach → Closer → Cliente
                                                ↗
                              DataAnalyst (monitoreo + inteligencia)
```

## Equipo de agentes

| Agente | Rol | Responsabilidad |
|--------|-----|-----------------|
| CEO | Coordinador | Asigna tareas, aprueba propuestas, monitorea pipeline |
| Scout | Prospectador | Encuentra negocios locales sin presencia digital |
| Qualifier | Analista SEO | Califica prospectos (score 1-10), genera diagnóstico HTML y propuesta |
| WebDesigner | Diseñador web | Crea propuestas web premium en HTML y las publica en Surge.sh |
| Outreach | Comercial | Genera materiales, envía mensaje 1 por email (draft) y WhatsApp (automático) |
| **Closer** | **Cerrador de ventas** | **Seguimiento mensajes 2 y 3, manejo de respuestas con IA, cierre consultivo** |
| DataAnalyst | Analista de datos | Monitorea pipeline, investiga mercados, analiza rendimiento |

## Flujo de trabajo

1. CEO recibe solicitud: "prospectar {giro} en {ciudad}"
2. Scout investiga y genera lista de prospectos con datos de contacto
3. Qualifier analiza presencia digital, genera score y diagnóstico HTML
4. WebDesigner crea propuesta web + página de diagnóstico en Surge.sh
5. Outreach genera materiales, envía mensaje 1 (email draft + WhatsApp automático)
6. **Closer recibe ticket 3 días después, envía mensaje 2 de seguimiento**
7. **Closer envía mensaje 3 (día 7) si no hay respuesta, o maneja respuestas con IA**
8. **Closer escala a CEO si hay interés, o cierra el prospecto**
9. DataAnalyst genera reportes semanales de pipeline y recomendaciones

## Mercado objetivo

Negocios locales en México (estéticas, dentistas, restaurantes, abogados, inmobiliarias, veterinarias, etc.)
que no tienen presencia digital o la tienen deficiente.
