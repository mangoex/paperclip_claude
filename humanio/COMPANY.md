# Humanio Marketing

Agencia de marketing digital enfocada en negocios locales en México. Operamos un pipeline automatizado de prospección, calificación SEO, diseño web y outreach comercial.

## Pipeline

```
Scout → Qualifier → WebDesigner → Outreach → Cliente
                ↘ DataAnalyst (monitoreo + inteligencia)
```

## Equipo de agentes

| Agente | Rol | Responsabilidad |
|--------|-----|-----------------|
| CEO | Coordinador | Asigna tareas, aprueba propuestas, monitorea pipeline |
| Scout | Prospectador | Encuentra negocios locales sin presencia digital |
| Qualifier | Analista SEO | Califica prospectos (score 1-10), genera diagnóstico HTML y propuesta |
| WebDesigner | Diseñador web | Crea propuestas web premium en HTML y las publica en Netlify |
| Outreach | Comercial | Genera PDFs, envía emails (draft) y WhatsApp (automático tras aprobación) |
| DataAnalyst | Analista de datos | Monitorea pipeline, investiga mercados, analiza rendimiento de outreach |

## Flujo de trabajo

1. CEO recibe solicitud: "prospectar {giro} en {ciudad}"
2. Scout investiga y genera lista de prospectos con datos de contacto
3. Qualifier analiza presencia digital, genera score y diagnóstico HTML
4. WebDesigner crea propuesta web + página de diagnóstico en Netlify
5. Outreach genera PDFs, draft de email y envía WhatsApp con la propuesta
6. DataAnalyst genera reportes semanales de pipeline y recomendaciones

## Mercado objetivo

Negocios locales en México (estéticas, dentistas, restaurantes, abogados, inmobiliarias, etc.) que no tienen presencia digital o la tienen deficiente.
